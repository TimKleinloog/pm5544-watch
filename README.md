# PM5544 watch

</div>

This repo the code to create a watch face for Garmin smartwatches based on the classic Philips PM5544 TV test pattern.

![pm5544](./images/PM5544.svg)

</div>

## Device support
Currently developed for Garmin Fenix 7X and will be published soon the the Garmin Connect IQ store. Contributions are welcome to support more devices.

## Development
Developed using [Garmin Connect IQ SDK](https://developer.garmin.com/connect-iq/sdk/). and [VSCode Monkey C extension](https://marketplace.visualstudio.com/items?itemName=garmin.monkey-c). For more information check the [Garmin developer docs](https://developer.garmin.com/connect-iq/connect-iq-basics/getting-started/).
